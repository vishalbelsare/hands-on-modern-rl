# 路线二：J(θ)——直接优化策略

上一节我们走了路线一：先学 $Q(s,a)$，再用 $\arg\max$ 选最好的动作。这条路在动作离散且数量少的时候（比如左推/右推、拉 A/拉 B）效果很好。但当动作空间变成连续的，$\arg\max$ 就失灵了——你不可能给每一个可能的力矩、每一个可能的 token 都算一个 $Q$ 值再取最大。

路线二提供了一种完全不同的思路：**跳过打分，直接学"在什么情况下该做什么"。**

## 从"打分再选"到"直接学行为"

还是走迷宫的例子。路线一的做法是给每条路挂牌子打分，然后选分数最高的。路线二的做法是：**不给路打分，而是反复走很多次迷宫。** 走到终点就加强沿途每个选择的信心，掉进陷阱就削弱。走得多了，好动作的概率自然上升，坏动作的概率自然下降。

这两种做法的区别可以用一个日常类比来理解：

- **路线一（Value-Based）**就像考试——你先做大量练习题（估计 Q 值），搞清楚每类题该怎么做，考试时选最有把握的答案。
- **路线二（Policy-Based）**就像学骑自行车——你不会先算"左转 15 度值 0.87 分"，而是直接反复练习，身体自然学会"感觉这样骑更稳"。

## 策略 π_θ：参数化的"行为手册"

在第 3.2 节中，我们定义了策略 $\pi$——给定状态输出动作的函数。现在我们要让策略变得**可学习**：用参数 $\theta$ 来定义它，写作 $\pi_\theta(a|s)$。

参数 $\theta$ 可以是神经网络的权重。训练开始时，$\theta$ 是随机的，策略行为混乱。随着训练的进行，$\theta$ 不断调整，策略越来越好——好动作的概率上升，坏动作的概率下降。

## J(θ)：衡量策略好坏的目标函数

那么怎么衡量一个参数化策略 $\pi_\theta$ 的好坏？用目标函数 $J(\theta)$：

$$J(\theta) = \mathbb{E}_{\pi_\theta} \left[ G_t \right]$$

$J(\theta)$ 的含义是：如果智能体遵循策略 $\pi_\theta$ 行动，平均能拿多少分。$J(\theta)$ 越高，策略越好。

训练的终极目标就是找到让 $J(\theta)$ 最大的参数：

$$\theta^* = \arg\max_\theta \, J(\theta)$$

这和路线一的逻辑形成鲜明对比：

| | 路线一（Value-Based） | 路线二（Policy-Based） |
| --- | --- | --- |
| 学什么 | $Q(s,a)$：每个动作值多少分 | $\pi_\theta(a|s)$：看到什么做什么 |
| 怎么决策 | $\arg\max_a Q(s,a)$（选最高分） | 直接从 $\pi_\theta$ 采样 |
| 目标 | 找到最优 $Q^*$ | 找到最优 $\theta^*$ |
| 直觉 | 先做练习题，考试选最有把握的 | 反复练习，形成肌肉记忆 |

## 这条路线的代表算法

沿着"直接优化策略"这条路，主要的算法演进是：

| 算法 | 核心思路 | 特点 |
| --- | --- | --- |
| REINFORCE | 用完整轨迹的回报调整动作概率 | 最简单的策略梯度，但方差很大 |
| Actor-Critic | 加入评委网络降低方差 | 将在第 6 章详细介绍 |
| PPO | 约束每步更新幅度，稳定训练 | 工业界主流，LLM 对齐的标准算法 |

这条路线的优势是**天然支持连续动作空间**和**擅长探索**（随机性策略总有概率尝试新动作）。但它的短板是**打分不够准**——策略梯度方法的方差很大，同一种策略跑两次，梯度估计可能天差地别，就像同一个学生两次考试分数波动很大一样。

第 5 章将深入这条路线，从最基础的 REINFORCE 开始，亲手体验策略梯度的工作方式和高方差问题。

## 两条路线的对照

现在我们已经看到了两条截然不同的路线。它们要解决的是同一个问题——找到最优策略 $\pi^*$——但思路完全不同：

- **路线一**先搞清楚每个动作值多少分（学 $Q$），再选最高的（$\arg\max$）。打分准，但不擅长探索。
- **路线二**跳过打分，直接学"该做什么"（优化 $J(\theta)$）。擅长探索，但打分不准。

一个自然的想法是：能不能把两条路线的优点拼在一起？用路线一的方法打分，用路线二的方法选动作？这正是 Actor-Critic 架构的核心思想，我们将在第 6 章中详细介绍。
