# D.2 概率、期望与随机估计

强化学习里的数据来自随机交互——策略随机选动作，环境随机给反馈。要理解这些随机性，就需要概率论。这一节按概率论的自然顺序展开：先定义样本空间、事件和随机变量，再讲概率、条件概率、期望和方差，最后进入蒙特卡洛估计、重要性采样和贝尔曼期望方程。
